# 🔇 Mute Me ! — Minimal Chrome Extension

## What it does:
- Lists tabs in the current window with an interactive checkbox for each.
- Toggle a checkbox to mute or unmute that specific tab.
- Beautiful, modern UI with smooth animations.

Install (developer mode):
1. Open `chrome://extensions` in Chrome.
2. Enable "Developer mode" (top-right).
3. Click "Load unpacked" and select this `Mute me !/Extension` folder.
4. Click the extension icon to open the popup and mute/unmute tabs.

## Features:
- 🎨 Modern, aesthetic design with gradient UI
- ⚡ Fast and responsive
- 🔒 Uses Manifest V3 for enhanced security
- 🎵 If a tab is playing audio, muting stops sound for that tab only

## Technical Details:
- **Manifest Version:** V3
- **Permissions:** `tabs`
- **No data collection** — everything runs locally
