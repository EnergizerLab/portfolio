Chrome extension project

Mute noisy tabs fast.

Mute Me is a Manifest V3 Chrome extension that lists tabs in the current browser window and toggles audio state from a compact popup.

Browser utility

MV3

Built around Chrome tab permissions, local popup UI, and direct user control.

Small tool, clear behavior.

The extension keeps one job in focus: show the open tabs and make muting predictable.

What it does

The popup reads the tabs in the active Chrome window, shows their current audio state, and lets the user switch each tab between muted and unmuted.

  • Lists open tabs in the current window.
  • Provides individual mute and unmute controls.
  • Uses Manifest V3 extension structure.
  • Keeps the interface compact enough for a browser popup.

How to install locally

Load the unpacked extension from the copied extension folder in this portfolio repo.

  1. Open Chrome and visit chrome://extensions.
  2. Enable Developer mode.
  3. Select Load unpacked.
  4. Choose Portfolio/Mute me !/Extension.
  5. Open the extension popup and control tab audio.

What I learned

This project connected the Chrome Tabs API with a focused popup interface. It also made permissions, local extension loading, and state updates more concrete than a static page alone.

Privacy shape

The extension is intended for local browser control. It does not need a server, account, or analytics pipeline to mute a tab.
